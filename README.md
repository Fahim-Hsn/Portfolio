# Fahim Hossain - Personal Portfolio

Hey there!  Welcome to the repository of my personal portfolio. I built this space to showcase my work, experiments, and journey as a developer bridging the gap between hardware embedded systems and modern web architectures. 

You can check out the live version of this portfolio here: **[fahimhsn.iam.bd](http://fahimhsn.iam.bd)**

---

##  What's Inside?

I wanted this portfolio to feel dynamic and alive, so it's not just a static page. Here are some of the key sections you'll find:

- **Hero Section:** A warm introduction with sleek, modern UI.
- **Skills Galaxy:** An interactive physics-based skill presentation using `matter.js`. 
- **Featured Projects:** A deep dive into my top projects, highlighting both software and hardware architectures (like IoT solutions, RFID access control, and GPS tracking).
- **Assignments:** A dedicated spot where I showcase academic and independent coursework, complete with screenshots and live previews.
- **Experience:** A timeline of my professional background and journey.
- **Contact:** Let's connect! Direct links to my GitHub, LinkedIn, and email.

##  Tech Stack

I built this project using some of my favorite modern tools:
- **Framework:** [Next.js](https://nextjs.org/) (React 19)
- **Styling:** CSS Modules with Glassmorphism aesthetics & [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for those buttery-smooth page transitions and hover effects.
- **Physics Engine:** [Matter.js](https://brm.io/matter-js/) (used for the interactive skills section!)
- **Icons:** `lucide-react` & `react-icons`
- **Language:** TypeScript 

##  Running it Locally

Want to poke around the code? Awesome! Here's how to get it running on your local machine:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Fahim-Hsn/Portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser and you're good to go!
